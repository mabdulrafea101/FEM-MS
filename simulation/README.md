# Cement Beam Simulation Module

This module implements the Finite Element Model (FEM) for simulating the natural frequency shifts in Fixed-Fixed Reinforced Concrete beams due to corrosion.

## Directory Structure

- **`src/`**: Source code for the simulation.
  - `fem_core.py`: The core physics engine implementing the Matrix Structural Analysis and Stiffness Reduction Method.
  - `generate_dataset.py`: Script to generate the synthetic dataset (`beam_vibration_dataset.csv`) using Latin Hypercube Sampling.
  - `visualize_results.py`: Script to generate plots of mode shapes and frequency trends.
- **`tests/`**: Validation scripts.
  - `validation.py`: Runs theoretical and experimental validation tests.
- **`data/`**: Contains the generated dataset.
  - `beam_vibration_dataset.csv`: The output dataset.
- **`outputs/`**: Generated graphs and other outputs.
  - `figures/`: Plots generated by `visualize_results.py`.
- **`logs/`**: Log files from the simulation runs.

## How to Run

1.  **Validation:**
    Run the validation tests to ensure the model is accurate.

    ```bash
    python3 simulation/tests/validation.py
    ```

2.  **Generate Dataset:**
    Generate the dataset for machine learning. This will create `data/beam_vibration_dataset.csv`.

    ```bash
    python3 simulation/src/generate_dataset.py
    ```

3.  **Visualize Results:**
    Generate plots to visualize the physics and the dataset distribution.
    ```bash
    python3 simulation/src/visualize_results.py
    ```
